# 内容素材评分器

这是一个面向短视频/口播素材评估场景的开源技能，用 5 维评分体系把"凭感觉拍"变成"按维度打分"。

它给每条素材一条清晰的决策记录：

- 够不够 A 级立即生产？
- 排第几优先级？
- 选哪个角度作为主角度？
- 哪些素材应该直接丢弃？

## 适合谁用

- 短视频创作者、口播博主、内容运营。
- 素材来源多（直播切片、连麦、用户原话、临时灵感），需要一套稳定筛选流程。
- 想要一份可复用、能横向比较的素材评分记录。
- 不想每次都凭感觉决定"这条拍不拍"。

## 快速开始

1. 复制 `assets/candidate-table-template.md` 到自己的笔记目录，改名 `materials.md`。
2. 在 Codex 或同款 AI 线程里贴上素材，先问"硬性排除能不能过"。
3. 跑完一轮 5 维打分后，把结果回写到 `materials.md`。
4. 想调整评分口径时，改 `references/score-rubric.md` 里的过滤规则和评分档位。

## 文件

- `README.md`：本文件。
- `LICENSE`：MIT 许可证。
- `SKILL.md`：可复用的技能说明。
- `assets/candidate-table-template.md`：素材候选表模板。
- `references/score-rubric.md`：5 维评分细则 + 可配置过滤规则。

## 推荐流程

1. 把素材贴进线程。
2. 先过 `speaker_filter` 和 `topic_filter`。
3. 30 秒快速排除 3 个问题。
4. 5 维打分，套加权公式。
5. 按评级 A/B/C 决定下一步：A 立即生产，B 排优先级，C 丢弃。
6. 把结果回写到 `materials.md`。
